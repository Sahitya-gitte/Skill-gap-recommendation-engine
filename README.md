# Skill Gap Recommendation Engine

## Overview
The Skill Gap Recommendation Engine is an AI/ML based web application that helps users identify missing skills required for their desired job roles. It compares the user's current skills with industry-required skills and recommends improvements.

## Objective
To bridge the gap between existing skills and required job market skills using Machine Learning techniques.

## Features
- Skill analysis based on user input
- Detects missing skills
- Recommends relevant skills to learn
- Displays skill match percentage
- Simple and interactive interface

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity

## Project Workflow
1. User enters current skills.
2. Selects desired job role.
3. System compares skills with required skills.
4. Missing skills are identified.
5. Recommendations are displayed.

## Dataset
Custom dataset containing:
- Job Roles
- Required Skills
- Recommended Skills
